# Commodity Price Chatbot 🤖

This is a simple rule-based chatbot built using Python.

## Features
- Intent detection (greeting, price query, goodbye)
- Commodity extraction (onion, potato, tomato)
- Dynamic responses based on user input

## Example
User: What is the price of onion?
Bot: The price of onion is ₹30 per kg.

## Tech Used
- Python
- Basic NLP (rule-based)

## Future Improvements
- Add more commodities
- Use ML-based NLP model
- Build UI using Streamlit
